关于我们

梦云智软件开发团队是在天津精耕科技发展有限公司的支持下成立的位于河北工业大学内的计算机学习实践团队.

团队成立于2014年12月份，成立之初得到计算中心实验室的大力支持。成立之初，由1名专任老师，3名研究生及5名本科生组成。至2019年8月，团队在队成员10人。90%为在读学生。历界团队成员都有着优异的表现，取得了广大同学、教师、用人单位的认同。

  • 一流的硬件设施
  • 一流的团队文化
  • 紧随国际前沿的计算机技术

学习路线

1

HTML CSS JAVASCRIPT

由最基础的HTML页面做起，学习网页布局。盒子与浮动流。了解一身的商业网的开发流程以及一些基础工具体的使用。
掌握基础的布局规则，掌握一般的配色方案。

  • 个人简介页面制作练习
  • 访大众点评页面制作练习
  • 管理系统页面制作练习
2

THINKPHP5入门实例教程

通过对教程的学习，了解B/S架构，了解面象对象在实际项目中的应用。了解MVC设计思想。清楚数据流，会使用GITHUB进行团队协作开发。可以使用TP5开发中小型管理系统。

  • 河北工业大学教务系统的制作练习
  • 梦云智开发团队出勤系统的制作练习
  • 基于微信的微课堂系统的制作练习
3

ANGULAR + SPRINGBOOT

了解前后台分离的开发思想。能使用angular框架快速开发前台WEB应用。使用spring-boot快速开发REST服务。了解AOP、IOC的基本原理以及实际的应用场景。在思想上由面向对象的编程思想，逐步向面向接口的编程思想过渡。

  • 参与alice学生管理系统的研发
  • 参与华软虚拟化项目的研发

项目介绍

动手动脑，成为优秀的软件工程师，为祖国的建设添砖加瓦！

天津职业技术师范大学经管学院绩效工资系统

该系统历时6个月。助力高院实现绩效工资管理。系统实现了科目细节的录入，建立了6种模型，解决了在绩效工资计算过程中，遇到的各类问题。
使原本半个月才能完成的工作，缩短到了1天。大幅降低了绩效工资计算的难度，提升了计算的准确度。

高职高专院校alice学生管理系统

结合高职高专的特点，在系统中加入了德育评分系统。并依托企业微信、微信，降低了老师与学生间沟通的成本。成绩管理模块通过建立计算模型，自动对录入的分数按设定的百分比进行计算。降低了学生管理的难度，提升了教师的工作效率。

计量器具管理平台

该平台以省为单位，对该省的所有器具信息、技术机构信息、检定能力信息、强检器具检定信息等所有的基础信息进行了电子管理。该系统设计了一套优秀的可靠的送检算法，该算法能够根据用户提交的器具、检定所的检定能力信息，自动计算器具的送检部门。在实际的使用中，得到广大用户的认同。
